== Description ==

Easily manage and create 301 redirects for your Wordpress website. A robust interface allows you to select from drop down menus, and validates your redirects for troubleshooting.

Created by Shawn Wernig [Eggplant Studios](http://www.eggplantstudios.ca/ "Eggplant Studios")

== Installation ==



1. Upload the `eps-301-redirects` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Begin adding redirects in the Settings > EPS 301 Redirects menu item.



== Frequently Asked Questions ==

= My redirects aren't working =

In the EPS 301 Redirects management page, please make sure your status code is 301. 

= My redirects aren't getting the 301 status code =

Your Request or Redirect URLS may be incorrect; please ensure that you are supplying valid URLs.
